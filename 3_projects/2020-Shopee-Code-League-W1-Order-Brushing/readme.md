#### Topic

Order Brushing 常見於電商平台上的商家，為了爭取出現在電商平台內搜尋結果的高位，重複高頻自行下單再退掉的現象。又稱「刷單」。
Shopee 定義刷單行為：
- 一小時內每人在相同商家下單大於 3 次
- 若有多人在相同商家下單都大於 3 次，以下單占比最高的列為可疑用戶
